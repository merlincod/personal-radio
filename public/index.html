À propos

Bienvenue sur votre lecteur radio personnalisé ! Ce projet, développé avec passion par Merlin's Code, vous offre une plateforme élégante et intuitive pour écouter vos musiques locales avec un style moderne et personnalisable.

Ce lecteur radio est conçu entièrement avec des technologies natives : HTML, CSS, et JavaScript, sans frameworks ni bibliothèques supplémentaires, pour garantir une expérience légère et entièrement personnalisable.

Stacks utilisées :

  • Frontend : HTML, CSS (responsive, animations), JavaScript natif (ES6+)
  • Backend : Node.js, Express.js pour la gestion des fichiers audio
  • Modules : music-metadata pour extraire les métadonnées MP3

Fonctionnalités principales :

  • Lecture et gestion de fichiers MP3 locaux
  • Interface minimaliste inspirée par NTS Radio
  • Visualisation des métadonnées : titre, artiste, album, durée
  • Contrôles modernes : lecture, pause, shuffle, repeat, volume
  • Adaptabilité mobile et desktop

Ce projet est une démonstration des possibilités offertes par les technologies web natives. N'hésitez pas à personnaliser ou améliorer cette plateforme selon vos besoins !
